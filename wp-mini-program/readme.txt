== Description ==
 
由 丸子小程序团队 基于 WordPress REST 创建小程序应用 API 数据接口。免费开源，实现 WordPress 连接小程序应用数据。<a href="https://developer.wordpress.org/rest-api/" taraget="_blank">WP REST API 使用帮助</a>。

== Installation ==
 
如何安装 Mini Program API.
 
e.g.
 
1. 下载 Mini Program API 插件，然后上传到 WordPress 网站后台插件安装
1. 通过 WordPress 网站后台菜单 插件 -  已安装插件 列表中，选择 Mini Program API 的启用，即可激活

== Frequently Asked Questions ==
为什么 API 地址请求 404
请检查站点是否支持伪静态，是否设置了固定链接以及 Rest API 是否被禁用
